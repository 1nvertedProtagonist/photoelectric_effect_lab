# PHYS 258 - An Investigation of the Wave-Particle Duality

This repository contains raw data, analysis code (Python + Jupyter) a lab logbook (PDF), the LaTeX report source code and its compiled PDF, generated figures from data analysis as well as illustrative figures for a  photoelectric effect lab. 

The goal was to explore and confirm the dual wave-particle nature of light by varying quantities in play inside a photodiode tube. Combinations of relationships between the dependent variables (current and stopping voltage) and the independent variables (photon flux density, illuminated surface area and incident energy of photons) were built and evaluated using statistical tools such as least-squares fitting, chi-2 goodness of fit test and weighted means of standard errors upon the mean. 

The findings of the experimentation agree with standard literature.

\## What’s in this repo

\- `raw_data/` — raw data files

\- `analysis/` — code/notebooks to analyze the data and generate figures 

\- `figures/photoelectric/` — figures used in the report (generated from analysis and illustrative purpose ones)

\- `report/` — LaTeX source (Overleaf export) and lab logbook PDF

\- `output/photoelectric\_report.pdf` — compiled report PDF
